# EasyPeak

Plataforma híbrida de ensino de inglês (Mobile, Web e Desktop) focada em gamificação, com 12 níveis e persistência de progresso em tempo real usando Supabase.

## Arquitetura
Projeto baseado nos princípios da **Clean Architecture**, dividindo responsabilidades em camadas claras (Domain, Data e Presentation).
O backend se suporta através do Supabase.
